## 1. プロジェクト概要

### 目的
SBI証券で実行可能なFX取引手法について、長期ヒストリカルデータでバックテストを実施し、最も優れた戦略を定量的に特定する。

### 評価指標
| 指標 | 説明 | 目標水準 |
|------|------|---------|
| Sharpe Ratio | 年率超過収益 / 年率標準偏差（無リスク金利=0） | > 0.5 |
| CAGR | 年率複利成長率 | > 5% |
| 最悪ドローダウン (Worst DD) | 期間中の最大峰値比下落率 | < -40% |

## 2. SBI証券 FX取引手法の全体像

### 2-1. 取引手法の分類

| 手法 | サービス名 | 種別 | レバレッジ | 最小単位 | 税制 |
|------|-----------|------|----------|---------|------|
| 店頭FX | SBI証券FX | 相対取引 | 最大25倍 | 100通貨 | 雑所得（総合） |
| 取引所FX | くりっく365 | 取引所 | 最大25倍 | 10,000通貨 | 申告分離20.315% |
| 積立FX | SBI証券 積立FX | 定期積立 | 最大25倍 | 100円〜 | 雑所得（総合） |
| 高スペックFX | SBI FXトレード | 相対取引 | 最大25倍 | 1通貨 | 雑所得（総合） |
| 取引所CFD | くりっく株365 | 取引所 | 最大25倍 | 1枚 | 申告分離20.315% |

### 2-2. バックテスト適合性評価

| 手法 | バックテスト可否 | 理由 |
|------|--------------|------|
| 店頭FX（SBI証券） | ✅ **可能** | 通貨ペアの価格データ（FRED/yfinance）でシミュレーション可能。スプレッド0.2銭をコストに組み込む |
| くりっく365 | ✅ **可能** | 同上。東京金融取引所データ一部参照可。スプレッドはやや広め（変動） |
| 積立FX（DCA） | ✅ **可能** | 月次固定額買付をシミュレーション。コスト: スプレッド0.2銭/回 |
| SBI FXトレード | ✅ **可能** | スプレッド0.18銭（最狭）。少額テスト後の移行先として活用 |
| くりっく株365 | ⚠️ **限定的** | 株価指数CFDのためFX戦略検証の主目的外 |

### 2-3. 重要な実運用上の差異
- **くりっく365**: 申告分離課税（20.315%固定）+ 3年損失繰越 → **長期投資・損益通算面で最有利**
- **店頭FX**: 総合課税 → 所得が高いほど税率上昇（最大55%）
- ⚠️ **バックテスト結果はスプレッドコストを反映するが、税コストは含まない**（実運用時は要調整）

## 4. 検証戦略の詳細仕様

### S1: 移動平均クロス（MAクロス）

| 項目 | 仕様 |
|------|------|
| 通貨ペア | USD/JPY（FRED 1971〜）|
| エントリー | 短期MA(50日) > 長期MA(200日): ロング / 逆: ショート |
| エグジット | 逆クロス |
| ポジションサイジング | 固定100万円投資 |
| コスト | スプレッド0.2銭/往復 |
| レバレッジ | 1倍（税制・リスク管理優先） |
| 期待Sharpe | 0.4〜0.6 |

### S2: ATRブレイクアウト（Donchian Channel）

| 項目 | 仕様 |
|------|------|
| 通貨ペア | USD/JPY, EUR/USD（複数） |
| エントリー | 直近20日高値超え: ロング / 直近20日安値割れ: ショート |
| エグジット | ATR(14) × 2 のトレイリングストップ |
| フィルタ | ATR圧縮フィルタ（現ATR < 過去50日ATR平均×0.7） |
| 期待Sharpe | 0.6〜0.9 |

### S3: キャリートレード

| 項目 | 仕様 |
|------|------|
| 通貨ペア | 金利差上位通貨ペア（月次更新） |
| エントリー | 高金利通貨買い / 低金利通貨売り、月次リバランス |
| エグジット | DDが-15%超: 50%縮小 / -25%超: 全撤退 |
| データ | FRED政策金利 + yfinance価格 |
| 期待Sharpe | 0.5〜1.3 |

### S4: RSI(14) 逆張り

| 項目 | 仕様 |
|------|------|
| 通貨ペア | USD/JPY |
| エントリー | RSI(14) < 30: ロング / > 70: ショート |
| エグジット | RSI中立域(45〜55)回帰 or ATR(14)×1.5ストップ |
| フィルタ | MA200より上でのみロング有効（トレンドフィルタ） |
| 期待Sharpe | 0.3〜0.5 |

### S5: ボリンジャーバンド(BB)平均回帰

| 項目 | 仕様 |
|------|------|
| 通貨ペア | USD/JPY |
| エントリー | +2σタッチ: ショート / -2σタッチ: ロング |
| エグジット | 中心線(20日MA)到達 or ±1σ |
| フィルタ | ADX(14) < 25（レンジ相場限定） |
| 期待Sharpe | 0.3〜0.6 |

### S6: 積立FX（ドルコスト平均法）

| 項目 | 仕様 |
|------|------|
| 通貨ペア | USD/JPY（SBI証券 積立FX想定） |
| 方式 | 毎月末 10万円分 USD買付 |
| エグジット | 保有継続（売却なし） |
| コスト | スプレッド0.2銭/月次 |
| 位置づけ | ベンチマーク（B&H戦略比較用） |
| 期待CAGR | 為替変動依存（過去実績: 長期プラス傾向） |

## 7. 重要な制約・注意事項

1. ⚠️ **룩ahead bias（先読みバイアス）厳禁**: エントリーは当日終値確定後の翌日始値で執行
2. ⚠️ **データ欠損処理**: FREDデータは土日・祝日欠損あり → 前日値で前方補完（`ffill`）
3. ⚠️ **スリッページ**: 日足戦略のためスプレッドのみ考慮。大口執行時のスリッページは含まない
4. ⚠️ **サバイバーシップバイアス**: 通貨ペア固定のため問題なし
5. ⚠️ **過学習リスク**: パラメータ最適化はOut-of-sample期間（直近10年）で必ず検証
