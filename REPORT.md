**対象期間:** 1990年〜2026年（最大36年）  
**対象ブローカー:** SBI証券 FX（スプレッド0.2銭〜）  
**評価軸:** シャープレシオ / CAGR / 最悪ドローダウン / カルマーレシオ  
**レバレッジ:** 1x（無レバ）  
**作成:** 2026-05-04

## エグゼクティブサマリー

**最重要結論: FXの単純戦略でSharpe 0.35を超えるのは極めて困難。**  
約36年・9戦略のバックテストで判明した核心は「**トレンド戦略のみが機能し、逆張り・平均回帰は機能しない**」という事実だ。最優秀戦略（S2: ATRブレイクアウト）でさえSharpe 0.345であり、無レバでS&P500（Sharpe ≈ 0.5〜0.6）を凌駕するのは難しい。

## 反直感的発見（重要）

### ⚠️ 発見1: USD/JPY キャリートレードは「罠」

**USD金利 > JPY金利** という事実だけを見るとUSDロングが有利に見えるが、実際はSharpe **-0.526**（最下位）。

**理由:** 
- 金利差が最大化した時期（2000年代、2022年以降）は同時に円安投機も極大化し、その後の「円高巻き戻し」で甚大な損失が発生する
- 2007〜2008年: USD/JPY は124円 → 88円（約30%の円高）でキャリー勢は壊滅
- 2022〜2024年: 一時150円超 → その後の急速巻き戻しリスク
- **教訓: 金利差収入より為替変動コストが大きい** のがUSD/JPYの構造的問題

### ⚠️ 発見2: AUD/USD キャリーは機能するが、USD/JPY と別物

同じ「キャリートレード」でもAUD/USDはSharpe 0.234。

**理由:**
- AUDは商品国通貨（鉄鉱石・石炭）＋高金利の組み合わせで、金利差がリスクプレミアムと整合している
- 安全資産逃避（リスクオフ）時もUSDが強まる分AUDが弱まり損失が出るが、基本方向はUSJ/JPYより安定
- **ただし** MaxDD -38.0%（2008年・2020年）は大きく、長期保有の精神的負荷は高い

### ⚠️ 発見3: 逆張り戦略（RSI・BB）は構造的にFXと相性が悪い

RSI逆張りSharpe -0.108、BB平均回帰Sharpe -0.343。

**理由:**
- FX（特にUSD/JPY）は長期に渡ってトレンドが継続する（70円台から160円台への多年間トレンド等）
- 逆張りは「トレンドが終わる」という仮定の下でしか機能しない
- **ATRストップを追加してもBB戦略のMaxDD -66.6%**（1990年代の円高トレンド中に深刻な損失）
- 教訓: **FXで逆張り戦略を使う場合、通貨ペア選択が最重要**（例: EUR/GBP等のレンジ通貨）

### ✅ 発見4: ATRブレイクアウトが唯一の「使える」戦略

全指標でバランスが取れた戦略は**S2のみ**：
- Sharpe 0.345（全戦略最高）
- MaxDD -22.1%（全戦略最低リスク）
- カルマーレシオ 0.051（最高）
- 57年間で510トレード（月平均0.7回）

**機能する理由:**
1. ATRが圧縮（ボラが低い）時のみエントリー → ブレイクが本物の可能性が高い局面を選択
2. トレーリングストップで利益を引き延ばし、損切りは素早く
3. ドンチャンチャンネルによる過去高値/安値ブレイクは「新しい均衡」への移行を捉える

## 戦略別詳細分析

### S1: MA クロス (50/200日) — 「無難だが非効率」

**Sharpe 0.162 / CAGR 1.1% / MaxDD -45.5%**

- ゴールデンクロス/デッドクロスは最も有名な手法だが、**シグナル遅延が致命的**
- 200日MAが反応するころには既にトレンドの大半は終わっている
- 57年間76トレードは年1.3回と少なく、長期ホールド中のドローダウン期間が長い（最長7,228日 = 約20年）
- **結論:** SBI FXでの実用性は低い。BnH以下の効率性

### S2: ATRブレイクアウト — 「唯一推奨」

**Sharpe 0.345 / CAGR 1.1% / MaxDD -22.1%**

- **最優秀戦略**。ドンチャン20日 × ATR圧縮フィルター × トレーリングストップの組み合わせ
- 年510/57 ≈ 8.9トレード（月0.7回）→ SBI FXの流動性・スプレッドで問題なし
- MaxDD -22.1%は最悪のリターン期間でも資産が78%以上確保される水準
- **弱点:** 年CAGR 1.1%は絶対水準として低い。レバレッジ3倍で約3.3%に向上可能
- **ブレイクイーブン条件:** 手数料・スプレッドコスト 0.2銭/取引はモデルに織り込み済み

### S3: キャリートレード — 「通貨選択が全て」

**USDJPY: Sharpe -0.526 / AUDUSD: Sharpe 0.234 / NZDUSD: Sharpe 0.166**

- **USDJPY:** 推奨しない（構造的失敗。詳細は発見1参照）
- **AUDUSD:** 最高CAGR 2.0%。長期保有向きだが-38%DDに耐える必要
- **NZDUSD:** AUDより安定感に欠けるが一定の収益性あり
- **実用上のポイント:** キャリーはくりっく365（申告分離課税20.315%）で保有が税務上有利

### S4: RSI逆張り — 「非推奨」

**Sharpe -0.108 / CAGR -0.2% / MaxDD -22.2%**

- トレンドフィルター（SMA200）を追加しても負のリターン
- MaxDD自体は小さいが勝てない戦略を実行する意義はない
- **唯一の可能性:** EUR/GBPなどの狭レンジ通貨ペアで再試験の価値あり（本バックテスト対象外）

### S5: BB平均回帰 — 「最も危険」

**Sharpe -0.343 / CAGR -1.5% / MaxDD -66.6%**

- ATRストップ追加後でも-66.6%。**資産が3分の1に減少する水準**
- 1990年代の円高トレンド（80円台）で平均回帰を繰り返し失敗し深刻な損失
- **厳禁:** USD/JPYでのBB逆張りは数値が証明している

### S6: 積立DCA — 「心理的に実用的」

**Sharpe 0.145 / CAGR 0.9% / MaxDD -44.3%**

- 毎月一定額のUSD購入。最もシンプルで実行が容易
- CAGR 0.9%はBnH（1.3%）を下回るが、**毎月の入金を「ドル転」という形で自動化**できる
- 円安局面では最大の恩恵を受けるが、円高局面では深刻な評価損
- **用途:** FX取引というよりも「外貨積立」として生活資産の一部に限定すべき

## SBI証券での実運用方針

### 推奨プラットフォーム比較

| 基準 | SBI FXトレード | くりっく365 |
|------|--------------|------------|
| スプレッド | 0.18銭（最安） | 変動制（広め） |
| スワップ課税 | 雑所得（最大55%） | **申告分離課税（20.315%固定）** |
| キャリー戦略 | △（課税効率悪） | ✅（推奨） |
| ブレイクアウト | ✅（コスト最小） | △（スプレッド広め） |
| 損益通算 | 不可 | **先物・FXと通算可能** |

### 実運用アクションプラン

#### ① メイン戦略: S2 ATRブレイクアウト（SBI FXトレード）

**運用手順:**
1. USD/JPY 1分足または日足でATR(14)とドンチャン(20)を確認
2. ATR < 過去20日ATR平均の場合のみエントリー対象とする
3. 20日高値ブレイクで買い、20日安値ブレイクで売り
4. エントリー後、ATR×2のトレーリングストップを設定
5. 最大ポジション: 証拠金の30%以内（実質レバレッジ3倍相当）

**推定年次収益（レバ3倍）:**
- CAGR: 約3〜4%（手数料控除後）
- MaxDD: 約-55〜-70%（レバ比例、最悪シナリオ）
- **リスク許容資産配分:** 総FX資産の50%以内

#### ② サブ戦略: AUD/USD キャリー（くりっく365）

**運用手順:**
1. AUD/USD の金利差がプラスの期間（AUD金利 > USD金利）のみ保有
2. 1ロット（10,000AUD）から開始、スワップポイントを月次で確認
3. ドローダウンが-20%超えたら50%縮小、-30%超えたら全決済
4. くりっく365で保有し、スワップ利益を申告分離課税で確定申告

**推定年次収益（無レバ）:**
- CAGR: 約2%（金利差収入＋為替変動）
- リスク: MaxDD -38.0%を許容できる規模で実施

#### ③ 除外戦略（実施しない）

| 戦略 | 除外理由 |
|------|---------|
| S3: USD/JPYキャリー | 構造的マイナス（Sharpe -0.526） |
| S4: RSI逆張り | FXとの相性が構造的に悪い |
| S5: BB平均回帰 | MaxDD -66.6%（許容不可） |

## リスク管理原則

### 1. ポジションサイジング

```
推奨FX総資産配分:
- SBI証券2,500万のうちFXへの割当: 最大500万（20%）
- ATRブレイクアウト用: 300万（レバ3倍 → 実効900万）
- AUD/USDキャリー用: 200万（無レバ）
```

### 2. ドローダウン管理

- 月次でポジション評価損益を確認
- ATRブレイクアウト: DD -30%到達でポジション50%縮小
- キャリー: DD -25%到達でDD回路遮断（全決済）
- 年間損失上限: -15%（FX割当分の絶対限度）

### 3. 相関リスク

- USD/JPYブレイクアウトとAUD/USDキャリーは**同方向リスク**を持つ場面がある
  （リスクオフ時: 円高→USD/JP Yショート収益、AUD安→キャリー損失）
- ポジション合計でのUSDネットエクスポージャーを±300万以内に管理

## 定量的到達目標と期待値

| シナリオ | 年間収益（500万FX資産） | 最悪ケース |
|---------|---------------------|----------|
| S2 × レバ3 | +15〜20万 | -150万（DD -30%到達） |
| AUD/USD キャリー | +4万（金利差） | -76万（DD -38%） |
| 合計 | **+19〜24万/年** | **-226万（同時最悪）** |
| 期待値（正規化） | +15万 | |

**比較: 同資産をS&P500 ETFに投資した場合の期待値**
- CAGR約7% → 500万×7% = **35万/年**
- MaxDD約-34%（2022年実績）

**結論: FX単純戦略は同リスクでS&P500の半分程度の期待値しかない。**  
FXに傾注するより、FXは為替ヘッジ・通貨分散の一手段と捉え、コア資産はS&P500等インデックスに置く方が合理的。

## 限界と今後の改善

### 本バックテストの限界

| 限界 | 影響度 | 補足 |
|------|-------|------|
| Close-only ATR（OHLCなし） | 中 | 実際のATR > 推定ATR → ストップが浅い可能性 |
| 日次データ（分足・時間足なし） | 中 | 日中のブレイクアウト・スリッページ未考慮 |
| スプレッド固定モデル | 低 | 2008年等の危機時はスプレッド拡大 |
| 限月ロールコスト未考慮 | 低 | くりっく365のロールコスト |
| 税金未計上 | 中 | 実効リターンは税後で20〜30%減少 |

### 改善候補（優先度順）

1. **S2のパラメータ最適化:** ドンチャン期間(10/20/40)× ATR期間(7/14/21)のグリッドサーチ
2. **通貨ペア拡張:** EUR/USD、GBP/USD でのATRブレイクアウト検証（USD/JPY依存の解消）
3. **時間軸変更:** 週次シグナル（日次より少ないトレード・ノイズ減少）の検証
4. **機械学習フィルター:** LightGBMでトレンド環境を判別し、トレンド局面のみS2実行
5. **レジームフィルター:** VIX、DXYを使ったマクロ環境フィルタリング

## 結論

1. **唯一の推奨戦略: S2 ATRブレイクアウト** — Sharpe 0.345、MaxDD -22.1%。FX単純戦略の実用上限に近い
2. **補助: AUD/USD キャリー** — CAGR 2.0%だが-38%DDリスクを許容できる場合のみ
3. **USD/JPYキャリーは絶対回避** — 金利差がプラスでも為替リスクが勝る（Sharpe -0.526）
4. **FXへの過度な傾注は非推奨** — 同リスクの米国株インデックスの方が期待リターンが高い
5. **実用的配分:** FXは総資産の10〜20%（500万以下）に限定、コア資産はS&P500/全世界株式
